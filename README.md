# Anthropic Course

**Master Claude — from your first prompt through advanced automation.**

A comprehensive, structured power-user course for Claude. Learn the Console, Claude Code CLI, and the API through **18 hands-on modules** with concepts, exercises, and quizzes.

No scattered tutorials. No marketing fluff. Just everything you actually need to know to go from zero to expert.

---

## What's Inside

📖 **18 Modules** organized by learning progression:
- Mental models and how Claude works
- Console, API, and models overview
- Everyday workflows (explore, debug, test, refactor)
- Advanced topics (memory, permissions, MCP, hooks, optimization)

✍️ **Concepts → Exercises → Quiz** format for each module:
- **Concepts:** What you need to understand (actual parameters, real workflows, product callouts)
- **Exercises:** Hands-on tasks you can do right now (exact commands, exact API calls, exact UI steps)
- **Quiz:** Test yourself before checking answers

🏷️ **Product-specific callouts:**
- `[Claude Code]` — CLI/app specific
- `[Console/API]` — platform.claude.com and API specific

## Quick Start

1. **Read** [`CLAUDE_COURSE.md`](./CLAUDE_COURSE.md) straight through, or jump to any module
2. **Do** the exercises in each module (you can do them in your own environment)
3. **Quiz** yourself to lock in the concepts
4. **Reference** the course anytime you need a refresher

No signup required. No videos. No login walls.

## How This Was Built

The course is generated from **all official Claude documentation**:
- 67 pages from [code.claude.com/docs](https://code.claude.com/docs)
- 39 pages from [platform.claude.com/docs](https://platform.claude.com/docs)

The docs are scraped, organized by topic, and structured into a cohesive learning path using Claude itself. This means the course stays up-to-date with new features.
